# ✈️ Airlines Information System (Data Management)

## 📌 Project Overview
The Airlines Information System is a data-driven project designed to manage and analyze airline-related data efficiently. This system focuses on airline operations such as flight details, aircraft information, routes, and scheduling. It does not include booking or reservation functionality.

This project demonstrates skills in data analysis, database management, and visualization.

---

## 🎯 Objectives
- Design a structured system for airline data management  
- Organize and manage large datasets efficiently  
- Analyze airline operations and identify patterns  
- Improve data accessibility and reporting  

---

## 🧰 Tools & Technologies Used
- SQL (MySQL)  
- Excel / CSV  
- Power BI  
- Python (optional)  

---

## 🗂️ Dataset Description
The dataset includes:
- Flight Details (Flight ID, Departure, Arrival, Duration)  
- Aircraft Information (Type, Capacity, Model)  
- Routes and Destinations  
- Scheduling Data  

---

## ⚙️ Step-by-Step Implementation

### 1. Data Collection
- Collected airline datasets from reliable sources  
- Verified data completeness  

### 2. Data Cleaning & Preprocessing
- Removed duplicates and null values  
- Standardized formats (dates, codes, locations)  
- Ensured data consistency  

### 3. Database Design
- Created relational database schema  
- Tables created:
  - Flights  
  - Aircraft  
  - Routes  
  - Schedules  
- Established relationships between tables  

### 4. Data Storage (SQL)
- Imported data into MySQL  
- Performed queries such as:
  - SELECT, WHERE, JOIN  
  - Filtering and aggregation  

### 5. Data Analysis
- Analyzed flight operations  
- Identified route patterns  
- Evaluated aircraft utilization  

### 6. Data Visualization (Power BI)
- Created dashboards showing:
  - Flight distribution  
  - Route analysis  
  - Aircraft usage  
- Used charts and KPIs for insights  

### 7. Insights & Findings
- High-frequency routes identified  
- Aircraft usage patterns analyzed  
- Scheduling trends observed
